# UPSC Study Atlas — India 1498–1947

An interactive historical map of the Indian subcontinent with AI-powered study assistance for UPSC Civil Services Examination (GS Paper 1 — Modern Indian History).

## Features
- 🗺 Interactive historical map with 55 territory periods (1498–1947)
- 📖 Time-indexed phase information for every empire and dynasty
- 🤖 AI Study Assistant powered by Gemini (ask questions about any map element)
- 📚 Notes library with all Spectrum chapters pre-loaded
- ✏️ UPSC-style quiz generator
- 📌 Add your own events and pin them to the map
- ✍️ Personal annotations on any event, territory, or city

## Files
- `index.html` — The main atlas application (~2MB, self-contained)
- `manifest.json` — PWA manifest (enables "Install App" on Android)
- `sw.js` — Service worker (enables offline use)
- `icon-192.png` / `icon-512.png` — App icons

## Setup
This is hosted on GitHub Pages. Access at:
`https://YOUR-USERNAME.github.io/upsc-study-atlas/`

## Usage
1. Open in Chrome on your Android tablet
2. Tap three-dot menu → **Install App** (or Add to Home Screen)
3. Enter your Gemini API key in the Ask AI tab (free from aistudio.google.com)
4. Your API key and personal notes are stored locally in your browser

## UPSC Coverage
- Modern Indian History: Europeans in India → Independence 1947
- All major empires: Mughal, Maratha, British, Sikh, Mysore, Vijayanagara, Deccan Sultanates
- 248+ events, battles, treaties, laws, movements, uprisings
- 67 historical cities with timeline context
- 10 Spectrum chapter notes pre-embedded
